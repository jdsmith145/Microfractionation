# Workflow context

The workflow supports a milligram-scale microfractionation strategy for active natural extracts. It connects three experimental data streams:

1. biological activity measured for crude extracts or HPLC fractions,
2. LC-MS or HRMS feature tables generated by MZmine and downstream annotation tools,
3. chromatographic timing information that links UPLC/MS features to HPLC microfractions.

The scripts are intended to make the computational part reproducible. GUI scripts help users configure runs and inspect columns. Core scripts are used for command-line execution, batch processing, and manuscript-supporting reproducibility.

The expected analysis path is:

1. identify active extracts with script 00,
2. run the MZmine-dependent complete/fraction processing and fraction-feature matching with script 01,
3. visualize chromatogram, fraction features, and activity/response values with script 02,
4. search known compounds in Wikidata with script 03,
5. predict feature fractions and map fraction response values to HRMS features with script 04,
6. summarize annotated HRMS tables with script 05,
7. use the notebooks as optional review companions for script 04 fraction prioritization and script 05 HRMS annotation summaries.

The scripts can also be used independently when a project already has the required input tables.

Script 01 contains the former standalone batch-preparation and fraction-matching logic. The public workflow intentionally exposes one MZmine-processing bridge so users do not need to decide between separate batch and matching tools.
